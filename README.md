# segundo-cerebro-notebooklm

## Descrição do projeto
Projeto de organização de conhecimento utilizando NotebookLM como um "segundo cérebro"
O objetivo foi desenvolver uma forma estruturada nde armazenar, organizar e consultar informações importantes. Portanto, permitindo que o usuário utilize a inteligência artificial como apoio para análise, aprendizado e tomada de decisões.

## Objetivos
- Criar um repositório de conhecimento estruturado.
- Utilizar inteligência artificial para auxiliar na análise de informações.
- Demonstrar como ferramentas de IA podem ser utilizadas para aprendizado e produtividade.
- Aplicar conceitos de organização de conhecimento em um ambiente digital.

## Ferramentas Utilizadas
- NotebookLM (Google)
- Inteligência Artificial
- Organização de documentos e fontes de conhecimento

## Metodologia
1. Seleção de conteúdos relevantes relacionados ao tema.
2. Inserção das fontes no NotebookLM.
3. Organização das informações para facilitar consultas.
4. Utilização da inteligência artificial para gerar insights, resumos e respostas baseadas nas fontes fornecidas.

## Resultados
O projeto demonstrou que ferramentas de IA podem ser utilizadas como um "segundo cérebro", auxiliando na organização do conhecimento e facilitando o acesso rápido a informações importantes.
